# DDS MVP

Изолированная MVP-версия расчета ДДС в статическом веб-приложении.

## Что нового

- Добавлено меню с 2 вкладками:
  - `Отчет ДДС`
  - `Статьи ДДС`
- На вкладке `Статьи ДДС` можно:
  - менять название статьи,
  - редактировать поля статьи,
  - удалять (помечать как удаленную) и восстанавливать статьи,
  - добавлять новые статьи.
- Изменения справочника сохраняются локально в браузере (`localStorage`).

## Возможности отчета

- Загрузка CSV операций.
- Расчет ДДС по статьям:
  - поступления,
  - выбытия,
  - чистый поток,
  - количество операций.
- Сводка по месяцам.
- Фильтрация по периоду и виду деятельности.
- Выгрузка итогового отчета в CSV.

## CSV операций

Обязательные колонки:

- `Дата`
- `Статья ДДС` (или `Статья`, `Фонд`)
- `Сумма`

Опционально:

- `Группа` (`Поступление` или `Выбытие`)

Если `Группа` не указана, направление операции определяется по знаку суммы
(плюс = поступление, минус = выбытие).

Пример: [dds-operations.sample.csv](/Users/katja/Documents/New%20project/dds-mvp/data/dds-operations.sample.csv)

## Запуск

### Вариант 1

Откройте [index.html](/Users/katja/Documents/New%20project/dds-mvp/index.html) в браузере.

### Вариант 2

```bash
cd dds-mvp
python3 -m http.server 8080
```

Откройте [http://localhost:8080](http://localhost:8080).

## Изоляция

- Все изменения и новые файлы только внутри папки `dds-mvp`.
- Вне `dds-mvp` ничего не изменяется.
