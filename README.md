# DDS MVP

Изолированная MVP-версия расчета ДДС в статическом веб-приложении.

## Что нового

- Добавлено меню с 4 вкладками:
  - `Отчет ДДС`
  - `Банки`
  - `Разнесение платежей`
  - `Статьи ДДС`
- На вкладке `Банки`:
  - есть блок `Настройка банков`,
  - список юрлиц изначально пустой (добавляете свои вручную),
  - можно добавлять новые юрлица и счета,
  - у каждого счета можно загрузить CSV-выписку,
  - выписка загружается с выбором месяца и сохраняется помесячно (новая загрузка не стирает другие месяцы),
  - банк можно мягко удалить и восстановить обратно,
  - удаление банка защищено многошаговым подтверждением (confirm + фраза `УДАЛИТЬ` + финальное confirm),
  - по каждому банку есть помесячная таблица (начало/конец месяца, поступления, выбытия, чистый поток),
  - под каждым юрлицом отображается мини-отчет по всем его счетам:
    - деньги на начало месяца,
    - деньги на конец месяца,
    - деньги на последнюю дату выписки,
    - сумма поступлений за месяц,
    - сумма выбытий за месяц,
  - есть сводка по каждому юрлицу и консолидировано по всем юрлицам.
- На вкладке `Разнесение платежей`:
  - показываются неразнесенные операции (без статьи/с неизвестной статьей),
  - можно вручную назначать статью через выпадающий список,
  - подсвечиваются проблемные строки,
  - есть выгрузка неразнесенных операций в CSV.
- На вкладке `Статьи ДДС` можно:
  - менять название статьи,
  - редактировать поля статьи,
  - удалять (помечать как удаленную) и восстанавливать статьи,
  - добавлять новые статьи.
- Изменения справочника сохраняются локально в браузере (`localStorage`).
- Настройки банков/юрлиц и загруженные выписки также сохраняются локально в браузере (`localStorage`).

## Возможности отчета

- Загрузка CSV операций.
- Расчет ДДС по статьям:
  - поступления,
  - выбытия,
  - чистый поток,
  - количество операций.
- Сводка по месяцам.
- Фильтрация по периоду и виду деятельности.
- Выгрузка итогового отчета в CSV.
- `Техническая операция` отображается отдельными строками, но не включается в нижний итог `Итого по ДДС`.
- Платежи без статьи учитываются в ДДС в блоке `03 Финансовая деятельность` как `Статья неизвестна`.

## CSV операций

Обязательные колонки:

- `Дата`
- `Сумма`

Опционально:

- `Статья ДДС` (или `Статья`, `Фонд`) — может быть пустой у неразнесенных платежей
- `Группа` (`Поступление` или `Выбытие`)
- `Контрагент`
- `Комментарий`

Если `Группа` не указана, направление операции определяется по знаку суммы
(плюс = поступление, минус = выбытие).

Пример: [dds-operations.sample.csv](/Users/katja/Documents/New%20project/dds-mvp/data/dds-operations.sample.csv)

## CSV банковской выписки

Обязательные колонки:

- `Дата`
- `Сумма` (или пара колонок `Дебет`/`Кредит`)

Опционально:

- `Баланс` / `Остаток`

Пример: [bank-statement.sample.csv](/Users/katja/Documents/New%20project/dds-mvp/data/bank-statement.sample.csv)

Поддерживаются типичные немецкие названия колонок: `Buchungstag`, `Wertstellung`, `Betrag`, `Umsatz`, `Saldo`, `Soll`, `Haben`.

## Запуск

### Вариант 1

Откройте [index.html](/Users/katja/Documents/New%20project/dds-mvp/index.html) в браузере.

### Вариант 2

```bash
cd dds-mvp
python3 -m http.server 8080
```

Откройте [http://localhost:8080](http://localhost:8080).

## Изоляция

- Все изменения и новые файлы только внутри папки `dds-mvp`.
- Вне `dds-mvp` ничего не изменяется.
