# DDS MVP

Изолированная первая версия справочника статей ДДС. Проект полностью автономен и не требует изменений в корне репозитория.

## Что внутри

- `index.html` — интерфейс MVP.
- `styles.css` — стили.
- `app.js` — данные и клиентская логика (поиск, фильтры, подсчет статусов, экспорт CSV).
- `data/dds-articles.sample.csv` — пример формата CSV.

## Функции MVP

- Таблица статей ДДС на основе предоставленного списка.
- Фильтры по:
  - текстовому поиску,
  - группе (`Поступление` / `Выбытие`),
  - виду деятельности,
  - статусу (`Активно`, `Неактуально`, `Удалить`, `Переименовать`).
- Цветовое выделение строк по статусам.
- Счетчики по текущей выборке.
- Экспорт отфильтрованного результата в CSV.

## Запуск

### Вариант 1 (самый простой)

Откройте `index.html` в браузере.

### Вариант 2 (локальный сервер)

```bash
cd dds-mvp
python3 -m http.server 8080
```

Затем откройте http://localhost:8080

## Принципы изоляции

- Все файлы MVP созданы только в папке `dds-mvp`.
- Вне `dds-mvp` ничего не изменялось.
