# DDS MVP

Изолированная MVP-версия расчета ДДС в статическом веб-приложении.

## Что нового

- Добавлено меню с 3 вкладками:
  - `Отчет ДДС`
  - `Разнесение платежей`
  - `Статьи ДДС`
- На вкладке `Разнесение платежей`:
  - показываются неразнесенные операции (без статьи/с неизвестной статьей),
  - можно вручную назначать статью через выпадающий список,
  - подсвечиваются проблемные строки,
  - есть выгрузка неразнесенных операций в CSV.
- На вкладке `Статьи ДДС` можно:
  - менять название статьи,
  - редактировать поля статьи,
  - удалять (помечать как удаленную) и восстанавливать статьи,
  - добавлять новые статьи.
- Изменения справочника сохраняются локально в браузере (`localStorage`).

## Возможности отчета

- Загрузка CSV операций.
- Расчет ДДС по статьям:
  - поступления,
  - выбытия,
  - чистый поток,
  - количество операций.
- Сводка по месяцам.
- Фильтрация по периоду и виду деятельности.
- Выгрузка итогового отчета в CSV.
- `Техническая операция` отображается отдельными строками, но не включается в нижний итог `Итого по ДДС`.
- Платежи без статьи учитываются в ДДС в блоке `03 Финансовая деятельность` как `Статья неизвестна`.

## CSV операций

Обязательные колонки:

- `Дата`
- `Сумма`

Опционально:

- `Статья ДДС` (или `Статья`, `Фонд`) — может быть пустой у неразнесенных платежей
- `Группа` (`Поступление` или `Выбытие`)
- `Контрагент`
- `Комментарий`

Если `Группа` не указана, направление операции определяется по знаку суммы
(плюс = поступление, минус = выбытие).

## Запуск

### Вариант 1

Откройте [index.html](/Users/katja/Documents/New%20project/dds-mvp/index.html) в браузере.

### Вариант 2

```bash
cd dds-mvp
python3 -m http.server 8080
```

Откройте [http://localhost:8080](http://localhost:8080).
