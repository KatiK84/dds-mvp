# DDS MVP

Изолированная MVP-версия расчета ДДС в статическом веб-приложении.

## Что делает

- Загружает CSV операций (из 1С или выгрузки с похожей структурой).
- Строит отчет ДДС по статьям:
  - поступления,
  - выбытия,
  - чистый поток,
  - количество операций.
- Показывает сводку по месяцам.
- Фильтрует отчет по периоду и виду деятельности.
- Выгружает итоговый отчет в CSV.
- Содержит справочник статей ДДС как вспомогательный блок.

## Структура CSV операций

Обязательные колонки:

- `Дата`
- `Статья ДДС` (или `Статья`, `Фонд`)
- `Сумма`

Опционально:

- `Группа` (`Поступление` или `Выбытие`)

Если `Группа` не указана, направление операции определяется по знаку суммы
(плюс = поступление, минус = выбытие).

Пример лежит в [data/dds-operations.sample.csv](/Users/katja/Documents/New%20project/dds-mvp/data/dds-operations.sample.csv).

## Запуск

### Вариант 1

Откройте [index.html](/Users/katja/Documents/New%20project/dds-mvp/index.html) в браузере.

### Вариант 2

```bash
cd dds-mvp
python3 -m http.server 8080
```

Откройте [http://localhost:8080](http://localhost:8080).

## Изоляция

- Все изменения и новые файлы только внутри папки `dds-mvp`.
- Вне `dds-mvp` ничего не изменяется.
